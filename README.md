## Overview

This repository hosts the official **Terms of Service** and **Privacy Policy** for **Bluu**, a free multi-feature Discord bot built for communities. Both pages are hand-crafted with premium visual effects ‚ÄĒ no frameworks, no dependencies, just raw WebGL and vanilla JavaScript.

<br />

## Pages

| Page | URL | Description |
|---|---|---|
| Terms of Service | `/terms.html` | Usage rules, conduct policy, and limitations |
| Privacy Policy | `/privacy.html` | Data collection, storage, and user rights |

<br />

## Visual Features

Every detail is intentional.

- **WebGL Silk Shader** ‚ÄĒ a real-time fragment shader ported from open-source, rendering a flowing fabric-like background unique to each page
- **Variable Proximity Title** ‚ÄĒ individual letters respond to cursor position, morphing font weight from 200 to 900 in real time using CSS `font-variation-settings`
- **Border Beam** ‚ÄĒ a conic gradient light that travels around card borders on hover, driven by a registered CSS `@property` animation
- **3D Card Tilt** ‚ÄĒ cards tilt on the X and Y axis relative to cursor position using CSS `perspective` transforms
- **ShinyText Badge** ‚ÄĒ a light sweep animation across the badge using animated `background-position`
- **Magnetic Button** ‚ÄĒ the navigation button is pulled toward the cursor using JS-driven translate offsets
- **Spinning Accent Ring Cursor** ‚ÄĒ a custom cursor with a small glow dot and a lagging ring with a conic gradient border that rotates continuously
- **Fade-up Animations** ‚ÄĒ each section enters the viewport with a smooth upward reveal via `IntersectionObserver`
- **Film Grain Overlay** ‚ÄĒ a subtle SVG fractal noise texture adds depth and prevents the design from feeling flat
- **Custom Scrollbar** ‚ÄĒ thin, minimal, matches the dark theme
