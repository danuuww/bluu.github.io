## Overview

Official **Terms of Service** and **Privacy Policy** pages for **Bluu**. Built entirely with raw WebGL and vanilla JavaScript, no frameworks, no external dependencies.

<br />

## Pages

| Page | Path | Description |
|---|---|---|
| Terms of Service | [`/terms.html`](https://danuuww.github.io/terms.html) | Usage rules, conduct policy, and liability |
| Privacy Policy | [`/privacy.html`](https://danuuww.github.io/privacy.html) | Data collection, storage, and user rights |

<br />

## Visual Implementation

| Effect | Description |
|---|---|
| WebGL Silk Shader | Real-time GLSL fragment shader, unique color per page |
| Variable Proximity Title | Each letter morphs font weight 200 to 900 based on cursor distance |
| Border Beam | Conic gradient light arc traveling the card border on hover |
| 3D Card Tilt | Cards tilt on both axes relative to cursor via CSS `perspective` |
| Accent Ring Cursor | Glow dot and lagging ring with rotating conic gradient border |
| Magnetic Button | Navigation button pulled toward cursor via JS translate offsets |
| ShinyText Badge | Light sweep animation via animated `background-position` |
| Fade-up Sections | Upward reveal on scroll via `IntersectionObserver` |
| Film Grain Texture | SVG fractal noise overlay for depth |

<br />
